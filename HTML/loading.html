Loading & Feedback — elegant, performant loaders

Copy-ready loading components for spinners, inline loaders, skeletons, progress bars and fancy animated placeholders. Each demo includes the full code for easy copy-paste.

Tip
Use CSS transforms and opacity where possible — they are GPU-friendly and smooth.

Gradient Ring Spinner

Smooth circular spinner — pure CSS

<!-- Gradient Ring Spinner -->
<div class="ring"></div>

<!-- CSS -->
.ring{width:72px;height:72px;border-radius:50%;position:relative}
.ring::before{
  content:'';position:absolute;inset:0;border-radius:inherit;
  border:8px solid rgba(255,255,255,0.06);border-top-color:var(--primary-1);
  animation: ringSpin 1.1s linear infinite;
}
.ring::after{ content:''; position:absolute; inset:10px; border-radius:50%; background: linear-gradient(90deg,#071426,#0b2136); }
@keyframes ringSpin{ to{ transform: rotate(360deg);} }

3D Cube

Spinning 3D cube using CSS transforms

<!-- 3D Cube -->
<div class="cube-scene">
  <div class="cube">
    <div class="face f-front"></div>
    <div class="face f-back"></div>
    <div class="face f-right"></div>
    <div class="face f-left"></div>
    <div class="face f-top"></div>
    <div class="face f-bottom"></div>
  </div>
</div>

<!-- CSS -->
.cube-scene{width:90px;height:90px;perspective:600px}
.cube{width:100%;height:100%;position:relative;transform-style:preserve-3d;animation:cubeSpin 3s linear infinite}
@keyframes cubeSpin{from{transform:rotateX(0) rotateY(0)}to{transform:rotateX(360deg) rotateY(360deg)}}
